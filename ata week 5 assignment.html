Algorithm Frequency Analysis Data Processing

Problem Statement

Create an algorithm to count the frequency of each letter (A-Z) in a text document. Your solution should ignore case (count 'a' and 'A' as the same letter) and ignore any non-alphabetic characters.

Input: "Hello, World!"

Output: {'H':1, 'E':1, 'L':3, 'O':2, 'R':1, 'W':1, 'D':1}

Key Insight

This problem requires us to parse text, filter characters, and count occurrences. It's a classic example of frequency analysis, which has applications in cryptography, data compression, and natural language processing.

 Part 4: Algorithm Design

                    
The step-by-step process for counting letter frequencies:

                    
                    

                        

                            
                            
Algorithm Flowchart

                            
1. Initialize an empty dictionary to store letter counts

                            
2. For each character in the text:

                            
   a. Check if it's an alphabetic character

                            
   b. If yes, convert to uppercase

                            
   c. Increment the count for this letter in the dictionary

                            
3. Return the final frequency dictionary

                        

                    

                    
                    

                        
 Time & Space Complexity

                        
Time Complexity: O(n) where n is the length of the text. We need to process each character once.

                        
Space Complexity: O(1) because there are at most 26 letters in the English alphabet, so our dictionary will have a constant maximum size.

                    

                    
                    
Pseudocode:

                    

                        

                            Letter Frequency Analysis
                            Pseudocode
                        

                        
FUNCTION CountLetterFrequency(text):
    Initialize empty dictionary frequency
    
    FOR each character c in text:
        IF c is an alphabetic character:
            Convert c to uppercase
            IF c exists in frequency:
                Increment frequency[c] by 1
            ELSE:
                Set frequency[c] to 1
    
    RETURN frequency
                        

                    

                

            

            
            

                

                    
 Part 5: Implementation

                    
Let's implement our algorithm in Python:

                    
                    

                        

                            Letter Frequency Analysis
                            Python
                        

                        
def count_letter_frequency(text):
    """
    Count the frequency of each letter (A-Z) in the text.
    Ignores case and non-alphabetic characters.
    
    Args:
        text (str): The input text to analyze
        
    Returns:
        dict: A dictionary mapping uppercase letters to their frequencies
    """
    frequency = {}
    
    for char in text:
        # Check if character is alphabetic
        if char.isalpha():
            # Convert to uppercase for case-insensitive counting
            char = char.upper()
            
            # Increment count in dictionary
            if char in frequency:
                frequency[char] += 1
            else:
                frequency[char] = 1
    
    return frequency

# Example usage
if __name__ == "__main__":
    sample_text = "Hello, World!"
    result = count_letter_frequency(sample_text)
    print(result)
    # Output: {'H': 1, 'E': 1, 'L': 3, 'O': 2, 'R': 1, 'W': 1, 'D': 1}
                        

                    

                    
                    
Alternative Implementation with Counter

                    

                        

                            Using Python's Counter
                            Python
                        

                        
from collections import Counter

def count_letter_frequency_with_counter(text):
    """
    Count the frequency of each letter (A-Z) in the text using Counter.
    Ignores case and non-alphabetic characters.
    
    Args:
        text (str): The input text to analyze
        
    Returns:
        dict: A dictionary mapping uppercase letters to their frequencies
    """
    # Filter for alphabetic characters and convert to uppercase
    letters = [char.upper() for char in text if char.isalpha()]
    
    # Use Counter to count occurrences
    frequency = Counter(letters)
    
    return dict(frequency)

# Example usage
if __name__ == "__main__":
    sample_text = "Hello, World!"
    result = count_letter_frequency_with_counter(sample_text)
    print(result)
    # Output: {'H': 1, 'E': 1, 'L': 3, 'O': 2, 'R': 1, 'W': 1, 'D': 1}
